# 初始提示词
``` markdown
你是一位专业的程序员，现在请你根据需求，帮我设计方案、人工确认、分步骤完成开发、执行测试、找我验收。

## 需求

我要第一时间获取一些热点(比如AI大模型的更新)，我不能依赖于人工手动搜索，而是利用工具自动发现指定的热点变化、或者发现最新的热点，并且及时给我发送通知，让我能够了解实时热点。
由于是一个比较轻量的小需求(工具类项目)，所以打算采用较为敏捷的开发方式，不用过多地人工介入和工程化。
具体来说，几个功能点:
* 用户手动输入要监控的关键词，当这个关键词相关的内容真的出现(注意利用 AI 识别假冒的内容)，第一时。间发送通知
* 每隔一段时间，自动搜集用户输入的指定范围(比如“AI编程”)内的热点，并目能够让用户看到。

产品的形式:
* 响应式兼容的 Web 页面
* 封装成 Agent Skills 技能，能够交给其他的 AI 来监控和发现热点

## 注意事项

1) 前端页面要足够独特,不要前篇一律
2) 你需要先完成网页版、确保功能正常使用之后，再开发Agent Skills
3) 项目需要利用AI来识别热点，请你利用OpenRouter来对接AI服务，帮我快速把AI接入到项目中。

## 获取信息的方案
一方面是利用网络搜索，比如在后端直接通过网页搜索的网址爬虫获取(不需要API、注意频率)，另外，你需要对接Twitter(X)的API来获取到最新的信息，比如从这里获取https://twitterapi.io/;注意，你需要同时从多个信息源获取信息，避免单一的信息源。

## 其余注意事项
1) 请你把需求和方案都沉淀为 Markdown 文档，然后再进行开发，注意，你必须利用 MCP 获取到最新的API 对接和技术实现方法
2) 你应该优先利用 Context7而不是 firecrawl mcp 来获取技术相关的文档
``` 

## 优化前端界面
A1 对于前端的设计可能千篇一律，所以最好使用前端美化技能(ui ux pro max)，并且人工指定UI风格，这样能让整个界面更新颖一些。

``` markdown
你需要利用前端美化技能(ui ux pro max)进行设计开发，并且必须采用Aceternity UI组件库(通过 Context7 插件来获取最新用法)，来增强网站的科技感，但是注意不要过于花里胡哨、影响了页面的正常加载或使用。
```
